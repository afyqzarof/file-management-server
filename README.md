# File Management Backend

[Link to Frontend](https://github.com/afyqzarof/file-management-client)

## To run

1. Ensure .env file is filled in according to .env.sample
2. Install dependencies:

```bash
npm install
```

3. Run migrations:

```bash
npm run migrate
```

4. Seed data:

```bash
npm run seed
```

5. Start development server:

```bash
npm run dev
```
